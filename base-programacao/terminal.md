# Comandos Terminal

1. Exibir o caminho atual

```bash
pwd
```

2. Navegar entre pastas

```bash
cd <caminho-pasta>
```

3. Listar arquivos e pastas

```bash
ls
```

4. criar pasta

```bash
mkdir <nome-pasta>
```

5. Apagar pasta

```bash
rmdir <nome-pasta>
```

6. Criar arquivo

```bash
touch <nome-arquivo>.<extensão>
```

7. Apagar arquivo

```bash
rm <nome-arquivo>.<extensão>
```

8. Copiar arquivos e pastas

```bash
cp
```

9.  Mover ou renomear arquivos e pastas

```bash
mv
```

10. Limpar Terminal

```bash
clear
# atalhos
ctrl + l
# ou
command + l
```

12. Fechar terminal

```bash
exit
```

13. Abrir o VSCode na Pasta atual

```bash
code .
```

14. Abrir o VSCode em uma pasta especifica

```bash
code <caminho-pasta>
```
